Cos'è CSS?

CSS è l'acronimo di Cascading Style Sheets. Ci sono 3 modi di utilizzare CSS:

  • style
  • Inline
  • External

In questo sito tratteremo il CSS esterno, quindi dobbiamo utilizzare un file esterno ad Html per poi collegarlo nel seguente modo. Esempio:

link type="text/css" rel="stylesheet" href="Comandi CSS.css"

Classe e id :

Se sul file HTML hai una classe, per modificarlo su CSS devi utilizzare ".nome classe".
Se invece vuoi modificare un id devi usare "#nome id"

Lo sfondo:

Per modificare i bordi in un titolo per esempio possiamo usare i seguenti comandi:

    background-color colore di sfondo
    background: linear-gradient(colore1 , colore2) più di un colore di sfondo
    background:no-repeat Non ripete i colori continuamente ( del background )
    background-attachment: fixed tutta la pagina avrà lo sfondo linear-gradient

    Per modificare i bordi del body per esempio possiamo usare i seguenti comandi:

      background-image:url(immagine.jpg) immagine di sfondo per farlo dobbiamo aprire un'altra pagina con l'immagine
      background-repeat: no-repeatL'Immagine di sfondo non viene ripetuto
      background-attachment:fixed tutta la pagina avrà lo sfondo
      background-position: centerLo sfondo verrà posizionato al centro
      background-size:cover Se allarghiamo o diminuiamo la grandezza della finestra, l'immagine si adatta

I margini:

I margini ( margins ) sono gli spazi tra gli elementi
per modificare i margini possiamo usare i seguenti comandi:

    margin-top l'elemento si sposta in basso ( si allontana dal mergine alto )
    margin-bottom l'elemento si sposta in alto ( si allontana dal mergine basso )
    margin-left l'elemento si sposta a destra ( si allontana dal mergine sinistro )
    margin-left l'elemento si sposta a sinistra ( si allontana dal mergine destro )
    margin: 0 l'elemento si attacca al margine sinistro
    margin-left:auto Sposta completamente l'elemento a sinistra
    margin-right:auto Sposta completamente l'elemento a destra


Il float:

Posiziona un elemento a destra o a sinistra di un container.
Si utilizzano le immagini principalmente per usare il float
I comandi sono :

    float:left Le scritte compaiono nella parte destra dell'immagine e l'immagine si sposterà a sinistra
    float:right Le scritte compaiono nella parte sinistra dell'immagine e l'immagine si sposterà a destra
    clear:left Smette di fare il float delle scritte, ma l'immagine resta nella posizione del float

Posizioni:

Per modificare le posizioni possiamo usare i seguenti comandi:


    position: static l'elemento ha una posizione default (non si muove)
    relative Possiamo muovere questo elemento relativo in alcuni punti rispetto l'origine
    absolute Rimuove dal flusso normale della pagina e viene posizionato in base al suo contenitore
    position:fixed Resta nella posizione del viewport, quindi anche se scorriamo verso il basso la pagina , l'elemento resterà per esempio nella parte in basso a sinistra dello schermo
    position:sticky Se scorriamo la pagina verso l'alto, quando superiamo l'elemento questo rimarrà attaccato nella parte in basso a sinistra della pagina ( se usiamo bottom , se usiamo top succede il contrario ) Se usiamo entrambi insieme ( bottom + top ), l'elemento si attaccherà nel punto dello schermo in base a dove scorriamo ( in alto o in basso).

Pseudo class:

Una parola chiave aggiunta a un selettore che specifica uno stato speciale dell'elemento selezionato.
Uno stato speciale può essere per esempio quando passiamo sopra l'elemento, quando clicchiamo l'elemento.
I comandi dello pseudo class con un link per esempio sono:


    ComandoSpiegazioneEsempio
    a:link Possiamo modificare per esempio il colore del link a:link{color:lawngreen}
    a:visitedquando visitiamo il sito la scritta può cambiare colorea:visited{color:grey}
    a:hoverQuando passiamo col cursore sopra l'elemento ( link o tabella ) cambia colorea:hover{color:tomato}
    a:activeQuando entriamo nel sito la scritta cambia colorea:active{color:yellow}
    li:nth-cild(numero elemento)Ci permette di modificare i singoli elementi di un elencoli:nth-cild(1){background-color:yellow}
    li:nth-cild(even-odd)Modifica gli elementi dispari(even)/pari(odd) dell'elencoli:nth-cild(even){background-color:skyblue}
